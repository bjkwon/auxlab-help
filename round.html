round

y = round ( x )
y = x.round ( )

Round to nearest decimal or integer
arg type Description Unit or Value scale
xSCAL, VCT or AUDIO n/a

Outputs
  • y is a "half-round-up" version of x.

Notes

Examples
  • round(2.6) → 3
  • round(2.4) → 2
  • round(-2.6) → -3
  • round(-2.4) → -2
