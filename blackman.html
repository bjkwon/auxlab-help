blackman

y = blackman ( x,alpha )
y = x.blackman ( alpha )

Apply a blackman window
arg type Description Unit or Value scale
xAUDIO or VCT Signal to apply the windowing to
alphaSCALα

Outputs
  • y is x with a blackman window applied.

Notes

Examples
  • blackman(ones(128),.4) or ones(128).blackman(.16): 128-pt blackman window with α=.16
  • blackman(dc(100)) or dc(100).blackman(.16): 100-ms blackman window α=.16
  • blackman(tone(500,100)) or tone(500,100).blackman(.16): blackman window α=.16 applied to a 500-Hz, 100-ms tone
