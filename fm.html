fm

y = fm ( freq1,freq2,mod_rate,duration,phase[=0] )

Generate a frequency-modulated tone signal
arg type Description Unit or Value scale
freq1SCAL FM swing from
freq2SCALFM swing to
mod_rateSCALhow many times the frequency will swing
durationSCALduration of the signal to generate
phaseSCALThe initial modulation phase between 0 and 1

Outputs
  • y is a frequency-modulated tone.
