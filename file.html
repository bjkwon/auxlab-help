file

y = file ( filename )

Read from a file
arg type Description Unit or Value scale
filenameTEXT the name of the file to read

Outputs
  • y is either a cell array (for a .txt file), or an audio signal (for an audio file)

Notes
  • Currently .txt, .wav. .mp3, .aiff files are allowed
  • For .txt file, each line of the file is read and made into each item of a cell array
