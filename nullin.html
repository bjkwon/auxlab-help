nullin

y = nullin ( x,p )
y = x.nullin ( p )

Insert time markers in an audio signal
arg type Description Unit or Value scale
xAUD audio signal
pTSEQ, VECT, or SCALTime sequence with desired tmarks. Values of the sequence are ignored.

Outputs
  • y is an audio signal from x with time markers as specified.

Notes
  • x can be a vector with time points in milliseconds.
