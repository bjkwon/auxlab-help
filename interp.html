interp

y = interp ( x,y,q )
y = x.interp ( y,q )

Convolute the two arrays
arg type Description Unit or Value scale
xVCT the x array
yVCTthe y array
qSCL or VCTthe proving value(s) that you want to interpolate

Outputs
  • y is the interpolated version of q. The length of y is the same as that of q.

Notes
  • x and y must be the same length.
  • q can be either a scalar or vector
