tsq_gettimes

y = tsq_gettimes ( x )
y = x.tsq_gettimes ( )

Retrieves time points of a time sequence
arg type Description Unit or Value scale
xTSEQ time_sequence_input

Outputs
  • y is a vector of time points of the input time sequence

Notes
  • If the input time sequence is relative time based, the output is the relative time point values, which can be used for tsq_settimes to specify relative time points (This is only a temporary hack; a better policy should be in place in the future. 3/11/2019)
