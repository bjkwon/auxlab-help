[FUNCTION NAME] zeros
[STATIC] 0
[FUNCTION DESCRIPTION]  Create a non-audio array of all zeros
[PARAMETERS]  n
[PARAM TYPE DELIM BY TAB]  SCAL
[PARAM DESCRIPTION--MULTILINES]  Size of array to create
[PARAM UNIT--MULTILINES]  n/a
[NOTES]  zeros generates a non-audio vector; silence generates an audio signal
[EXAMPLES]  
[SEE ALSO] ones 
[ALGORITHM]  
[OUTPUT ARGS BY TAB] y
[OUTPUT ARGS--MULTILINES] #q1 is a non-audio array with #p1 zeros
