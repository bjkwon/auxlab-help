write

y = write ( x,filename )
y = x.write ( filename )

Write data into a text file; write audio data to a .wav or .mp3 file
arg type Description Unit or Value scale
xAUD VCT or CELL data to write into the file
filenameTEXTthe name of the file to write

Outputs
  • y x

Notes
  • For .txt file, data is written row-wise, except for a cell array written on each row representing the item
