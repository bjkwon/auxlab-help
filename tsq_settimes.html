tsq_settimes

y = tsq_settimes ( x,time_points )
y = x.tsq_settimes ( time_points )

Retrieves time points of a time sequence
arg type Description Unit or Value scale
xTSEQ time_sequence_input
time_pointsVCTAn array of time point values;

Outputs
  • y is x, modified with the given parameter.

Notes
  • To update a millisecond-based input time sequence with relative time based time points, define a new time sequence with relative time base and use tsq_gettimes to pull up that relative time base and use it as the argument (This is only a temporary hack; a better policy should be in place in the future. 3/11/2019).
