tsq_setvalues

y = tsq_setvalues ( x,data )
y = x.tsq_setvalues ( data )

Sets the value(s) of the time sequence at each time point
arg type Description Unit or Value scale
xTSEQ time_sequence_input
dataMATXThe data (or the values) of the time sequence; Each group(each row) represents the data at each time point

Outputs
  • y is x, modified with the given parameter
