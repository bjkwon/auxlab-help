tsq_isrel

y = tsq_isrel ( x )
y = x.tsq_isrel ( )

Checks if the input, a time-sequence, is relative time based.
arg type Description Unit or Value scale
xTSEQ time_sequence_input

Outputs
  • y is 1 (true) or 0 (false)

Notes
  • If the input argument is not time sequence, it returns false.
