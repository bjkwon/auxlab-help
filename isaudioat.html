isaudioat

y = isaudioat ( x,t )
y = x.isaudioat ( t )

Checks if the audio signal is present (i.e., not "null") at the specified time point
arg type Description Unit or Value scale
xAUD audio signal
tSCALtime

Outputs
  • y is logical indicating whether the signal is not null at the point of x.

Notes

Examples
  • x=noise(300)@-20>>100;
  • x.isnull(50) → false.
  • x.isnull(200) → true.
