movespec

y = movespec ( x,freq )
y = x.movespec ( freq )
y = x -> freq

Shift the spectrum in the frequency domain
arg type Description Unit or Value scale
xAUD audio signal
freqVCThow much the signal is shifted by frequency

Outputs
  • y is audio with the same duration as x

Notes
  • This function shifts the frequencies all harmonic components; i.e., do not use this function to change the pitch.
