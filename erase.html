erase

y = erase ( obj,field_name )
y = obj.erase ( field_name )

Remove a field from a structure object
arg type Description Unit or Value scale
objSTRUCT A structure object
field_nameTXTThe string of the field

Outputs
  • y is the structure object

Notes

Examples
  • v=[]; v.fs=22050; v.ex = "something to get rid of";// A structure object is defined. fields: fs and ex
    v.erase("ex") // the field "ex" is removed.
