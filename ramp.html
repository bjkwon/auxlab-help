ramp

y = ramp ( x,dur_ramp )
y = x.ramp ( dur_ramp )

Smooth out beginning and ending portions of the audio signal; Apply the cosine square envelope
arg type Description Unit or Value scale
xAUDIO audio signal
dur_rampSCALramping duration (i.e., duration to smooth out)

Outputs
  • y is an audio signal with fade-in and fade-out.

Notes
  • Ramping is applied only to the portions specified by dur_ramp . Windowing functions, such as hann hamming or blackmand, apply the window for the whole duration.

Examples
  • tone(440,100).ramp(40)
