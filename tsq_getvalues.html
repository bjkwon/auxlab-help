tsq_getvalues

y = tsq_getvalues ( x )
y = x.tsq_getvalues ( )

Retrieves the value(s) of the time sequence at each time point
arg type Description Unit or Value scale
xTSEQ time_sequence_input

Outputs
  • y is a matrix, where the value(s) at time point is shown in each row.
