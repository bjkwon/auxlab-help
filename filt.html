filt

[y, final] = filt ( x,num,den[[=1]],initial[[=(empty)]] )
[y, final] = x.filt ( num,den[[=1]],initial[[=(empty)]] )

1-D digital filter
arg type Description Unit or Value scale
xAUD or VECT Input data
numVECTNumerator coefficients of rational transfer function
den[VECTDenominator coefficients of rational transfer function
initial[VECTInitial condition, a vector of max(num.length, den.length)-1

Outputs
  • y is filtered data.
  • final is the final condition after filtering

Notes
  • For a grouped or "2-D" array, filtering takes place for each group.
