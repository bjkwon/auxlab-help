eval

y = eval ( expression )

Execute AUXLAB expression in text string
arg type Description Unit or Value scale
expressionTXT String that contains a valid AUXLAB expression.

Outputs
  • y is null if there's no error in expression. y is the error message if there is an error.

Notes
  • expression may have multiple expressions.
  • If the execution of expression is successful, the output is NULL, and the variables are defined in the workspace as specified in expression.
  • If an error occurs in expression, eval gracefully returns with the error message as the output, instead of a raised exception. This ensures the continued execution of multiple expressions even if there is an error. This feature is useful when multiple expressions must be executed in sequence in a udf; for example, an automatic batch test.
